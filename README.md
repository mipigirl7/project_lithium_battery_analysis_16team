# 개요 - 태블로로 제조 데이터를 어디까지 보여줄 수 있는가?

---

🔋 **배터리는 단순한 부품이 아니라, 시간이 지날수록 성능이 변하는 “살아있는 데이터”와 같습니다.**

- 리튬 이온 배터리는 충전과 방전을 반복할수록 성능이 점차 저하됩니다.
- 같은 배터리라도 온도, 충전 상태, 내부 저항 변화에 따라 열화 속도가 달라질 수 있습니다.
- 데이터분석가는 이러한 변화 과정을 숫자만 보는 것이 아니라, **시계열 흐름과 상태 변화를 시각적으로 해석**하여 문제를 발견하고 개선 포인트를 제안합니다.

🔋 **당신은 배터리 제조사의 품질/분석 담당 데이터 애널리스트입니다.**

- 배터리 실험 데이터를 탐색하여 **배터리 열화 현황을 한눈에 보여주세요.**
- 어떤 배터리가 빠르게 성능이 저하되는지, 어떤 조건에서 열화가 심한지를 **데이터 근거로 설명해주세요.**

🔋 **제조 산업에서는 설비·부품·센서 데이터를 기반으로 품질 상태를 지속적으로 모니터링합니다.**

🔋 **우리는 이번 프로젝트를 통해 리튬 이온 배터리 데이터 기반 열화 현황 분석 및 잔존 수명 예측 대시보드 구축 프로젝트를 수행해 보겠습니다.**


# 배경

---

🔋 **리튬 이온 배터리란?**

- 리튬 이온 배터리는 전기차, ESS, 모바일 기기 등 다양한 산업에서 핵심적으로 사용되는 에너지 저장 장치입니다.
- 충전과 방전을 반복할수록 저장 가능한 용량은 감소하고, 내부 저항은 증가하게 됩니다.
- 따라서 제조 및 운영 관점에서는 “지금 얼마나 좋은 배터리인가?”뿐 아니라, **“앞으로 얼마나 더 쓸 수 있는가?”** 가 매우 중요합니다.

🔋 **이번 프로젝트에서 다루는 핵심 개념**

- **Charge(충전)**: 배터리에 에너지를 저장하는 과정
- **Discharge(방전)**: 배터리에 저장된 에너지를 사용하는 과정
- **Impedance(임피던스)**: 배터리 내부 저항 특성을 측정하는 과정
- **Capacity(용량)**: 배터리가 저장하고 방전할 수 있는 전하량
- **RUL(Remaining Useful Life)**: 현재 시점에서 앞으로 남은 사용 가능 수명

🔋 **왜 시각화가 중요한가요?**

- 배터리 데이터는 단순 표 형태보다, **사이클에 따른 변화 흐름**을 보는 것이 훨씬 중요합니다.
- 용량 감소, 온도 변화, 내부 저항 증가 같은 현상은 숫자 한 줄로는 잘 보이지 않습니다.
- 따라서 이번 과제는 모델 성능보다도, **배터리 상태 변화가 잘 보이는 대시보드 설계와 스토리텔링**이 핵심입니다.

🔋 **주요 이해 관계자 예시**

- 품질팀: 어떤 배터리가 비정상적으로 빨리 열화되는지 확인
- 생산기술팀: 온도나 시험 조건에 따라 성능 차이가 있는지 파악
- 연구개발팀: 향후 RUL 예측 모델 고도화를 위한 데이터 구조 이해
- 사업/임원: 배터리 성능 안정성과 리스크를 직관적으로 보고받기 원함
